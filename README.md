# Australian Weather Prediction using Machine Learning

## Objective
To predict whether it will rain tomorrow using a real-world Australian weather dataset.

## Dataset
- 147,000+ records
- 23 features
- Target Variable: RainTomorrow

## Data Preprocessing
- Missing value treatment (median, mode, forward & backward fill)
- Label encoding of categorical variables
- Feature correlation analysis
- Feature scaling using StandardScaler
- Class imbalance handling using SMOTE

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Naive Bayes
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

## Result
Achieved over 80% test accuracy with multiple classification models.

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Imbalanced-learn
